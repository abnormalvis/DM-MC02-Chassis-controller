# C8T6 电流采集固件需求文档（草案 v0.1）

## 1. 文档目的与范围

本文档定义 C8T6 单片机在 DM-MC02 底盘控制系统中的角色：**电流采样与上报**。

C8T6 负责：
- 采集电机驱动板的电流反馈信号
- 通过 CAN 总线将电流数据发送给 H723
- 接收 H723 下发的控制命令

## 2. 系统架构

```
┌─────────────┐                    ┌─────────────┐
│    C8T6     │ ──── CAN 总线 ────→│    H723     │
│ (电流采样)  │ ←─────────────────│ (控制环)    │
│             │   控制命令(0x102)  │             │
└─────────────┘                    └─────────────┘
      ↑                                  ↑
   ADC 采样                           电机控制
   电机A/B                            PWM 输出
```

## 3. 硬件连接

| 功能 | 引脚 | 说明 |
|------|------|------|
| CAN TX | 根据 C8T6 原理图 | 连接 H723 PD13 |
| CAN RX | 根据 C8T6 原理图 | 连接 H723 PD12 |
| 电流A采样 | ADC 通道 | 电机A电流反馈 |
| 电流B采样 | ADC 通道 | 电机B电流反馈 |

## 4. 功能需求

### 4.1 CAN 通信

1. **CAN 初始化**
   - 波特率：1 Mbps
   - 帧格式：Classic CAN
   - 标准 ID：0x101（电流数据）、0x102（控制命令）

2. **数据发送**
   - 发送频率：1kHz（默认，可配置）
   - 帧 ID：0x101
   - DLC：8 bytes

3. **命令接收**
   - 帧 ID：0x102
   - 支持命令：启动/停止采样、设置参数、零点校准

### 4.2 ADC 电流采样

1. **采样通道**：2 路 ADC，对应电机A和电机B的电流反馈
2. **采样频率**：默认 1kHz，与 CAN 发送频率同步
3. **数据格式**：
   - 有符号 16 位整数
   - 单位：0.01A
4. **校准**：
   - 支持零点校准（记录零点偏移）
   - 支持比例系数标定

### 4.3 时间戳机制

1. 使用 CAN 控制器内置时间戳
2. 时间戳单位：0.1ms（CAN 时钟分频后）
3. 每个数据帧携带采样时刻的时间戳

## 5. CAN 帧格式

### 5.1 电流数据帧 (0x101)

| 字节 | 字段 | 类型 | 说明 |
|------|------|------|------|
| 0-1 | current_a | int16 | 电机A电流，单位 0.01A |
| 2-3 | current_b | int16 | 电机B电流，单位 0.01A |
| 4 | seq | uint8 | 序号，0-255 循环 |
| 5 | status | uint8 | 状态位 |
| 6-7 | timestamp | uint16 | CAN 时间戳，单位 0.1ms |

### 5.2 控制命令帧 (0x102)

| 字节 | 字段 | 类型 | 说明 |
|------|------|------|------|
| 0 | cmd | uint8 | 命令码 |
| 1 | param | uint8 | 参数值 |

**命令码定义：**

| 命令码 | 功能 | param 说明 |
|--------|------|-------------|
| 0x00 | 请求采样 | 保留 |
| 0x01 | 设置参数 | 采样频率 (Hz) |
| 0x02 | 启动采样 | 保留 |
| 0x03 | 停止采样 | 保留 |
| 0x04 | 零点校准 | 保留 |

### 5.3 状态位定义

| 位 | 功能 |
|----|------|
| bit0 | 电机A过流标志 |
| bit1 | 电机B过流标志 |
| bit2 | C8T6异常标志 |
| bit3 | 采样数据有效标志 |

## 6. 通信时序

```
C8T6:
  采样 (t=0ms) → ADC转换 → CAN发送 (t≈0.1ms) → H723接收 (t≈0.3ms)
                      ↑
                  CAN时间戳 = 0ms时刻
```

**延迟分析：**
- ADC 采样延迟：~10μs
- CAN 发送延迟：~100μs
- 总延迟：< 200μs

## 7. 异常处理

| 异常 | 处理策略 |
|------|----------|
| CAN 总线断开 | 进入安全态，停止发送，LED 闪烁提示 |
| ADC 采样异常 | 标记 status bit2 异常位，继续发送无效数据 |
| 过流 | 标记 status bit0/1，发送告警帧 |

## 8. 调试接口

1. **串口调试**：可选，通过串口输出调试信息
2. **LED 指示**：
   - 正常运行：LED 常亮
   - CAN 通信异常：LED 闪烁
   - 过流告警：LED 快闪

## 9. 与 H723 端协议对应关系

| 项目 | C8T6 端 | H723 端 |
|------|---------|---------|
| CAN 外设 | FDCAN1 | FDCAN3 |
| 发送帧 ID | 0x101 | 0x101 (接收) |
| 接收帧 ID | 0x102 | 0x102 (发送) |
| 波特率 | 1 Mbps | 1 Mbps |
| 采样频率 | 1kHz (默认) | 1kHz (控制环) |
