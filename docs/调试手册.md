# DM-MC02 底盘控制调试手册

## 1. 硬件连接

### 1.1 核心板引脚分配

| 功能 | 引脚 | 说明 |
|------|------|------|
| CRSF接收 | UART7 (PG9/PG10) | 遥控器接收 |
| RS485 | USART2 (PD5/PD6), USART3 (PD8/PD9) | 上位机通信 |
| CAN3 | PD12/RX, PD13/TX | C8T6电流采集 |
| IMU | SPI2 (PC0-CS_ACC, PC1-MOSI, PC2-MISO, PC3-CS_GYRO) | BMI088 |
| PWM电机A | TIM1 (PA8-FWD, PA9-REV) | 电机A驱动 |
| PWM电机B | TIM2 (PA0-FWD, PA1-REV) | 电机B驱动 |
| LED | PC13 | 状态指示 |
| 蜂鸣器 | PC14 | 声音提示 |

### 1.2 调试接口

- **ST-Link**: SWDIO (PA13), SWCLK (PA14)
- **串口调试**: USART1 (PA9/PA10) - 115200 8N1

## 4. 各模块测试方法

### 4.1 CRSF遥控测试

**测试步骤**:
1. 连接ELRS接收器到UART7
2. 打开串口调试助手，查看日志
3. 推动遥控器摇杆，观察通道值

**预期输出**:
```
[CRSF] throttle: 0.50 steering: -0.25 mode: 1 estop: 0
[CRSF] connected: 1 frames: 1000 errors: 0
```

**故障排查**:
- 无数据: 检查UART7连接波特率(420000)
- 错误计数增加: 检查CRC校验

### 4.2 电机PWM测试

**测试步骤**:
1. 示波器探头接PA8(电机A正转)
2. 调用 `MotorControlTask_SetTarget(500, 0)`
3. 观察PWM波形

**预期结果**:
- 频率: 20kHz
- 占空比: 约50% (可调)

### 4.3 CAN通信测试 (C8T6)

**测试步骤**:
1. C8T6 上电，发送电流数据
2. 观察日志
3. 调用 `CANTask_GetCurrent(&ia, &ib)`

**预期输出**:
```
[CAN] current_a: 250 (2.5A) current_b: 248 (2.48A)
[CAN] connected: 1
```

### 4.4 IMU测试

**测试步骤**:
1. 连接BMI088 (SPI2)
2. 调用 `IMUTask_GetData(&data)`

**预期输出**:
```
[IMU] gx: 0.001 gy: 0.002 gz: 0.000
[IMU] ax: 0.12 ay: 0.05 az: 9.81
[IMU] roll: 0.01 pitch: 0.00 yaw: 0.00
```

### 4.5 USB/RS485通信测试

**测试步骤**:
1. 连接USB或RS485到上位机
2. 发送心跳包
3. 观察响应

**测试帧格式**:
```
// 下行: CMD_HEARTBEAT
55 AA 01 01 00 01 00 00 00 00 00 00 E8 03 [CRC16]

// 上行: STA_HEARTBEAT
55 AA 01 02 00 01 00 00 00 00 00 02 00 01 [CRC16]
```

## 6. 故障排查指南

### 6.1 常见故障

| 故障现象 | 可能原因 | 解决方法 |
|---------|---------|--------|
| 无PWM输出 | 刹车激活 | 检查CRSF/USB连接 |
| CRSF无效 | 超时 | 重启遥控器 |
| CAN无数据 | C8T6未启动 | 检查C8T6固件 |
| IMU无效 | SPI错误 | 检查BMI088连接 |
| USB枚举失败 | 驱动问题 | 安装ST驱动 |

### 6.2 安全机制

- **CRSF超时**: 100ms → 刹车
- **USB超时**: 100ms → ���车
- **CAN超时**: 100ms → 告警(不刹车)
- **急停**: 立即刹车
